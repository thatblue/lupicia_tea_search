# ルピシアのお茶飲み方サーチ

[ルピシア](https://www.lupicia.com/)のお茶の飲み方を調べるWebサイトのコードを作ろうとしているリポジトリです。

これはサードパーティーアプリケーションであり、[株式会社ルピシアホールディングス](https://www.lupicia.com/)とは無関係の個人がルピシアの製品・サービスを楽しく利用するために開発しているものです。  
このアプリケーションに関するお問い合わせ・フィードバックはissueへどうぞ。

開発者 @thatblue の推しお茶は[いもくりかぼ茶](https://www.lupicia.com/shop/g/g13339209/)です。秋季限定でちょっと油断すると買い逃してしまいますが、それもまた乙というものです。

This is a third party application and is developed by individuals unrelated to Lupicia Holdings Ltd. for the enjoyment of using Lupicia's products and services.  
For inquiries or feedback on this application, please contact issue.

Developer @thatblue 's recommendation is [Imo Kuri Kabo Cha](https://www.lupicia.com/shop/g/g13339209/). It is only available during the fall season, so if you are not careful, you may miss out on it, but that is also a good thing.
